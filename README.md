# Alphabet

![Alphabet](.doc/avatar.png)

Alphabet is an intelligent developer toolkit for 10x software engineering.

## Technical Specification

CLI application in the Bash programming language. 

## Installation

Simply clone the .git repository and source the Alphabet bash profile:

```sh
$ git clone https://github.com/w-ap/alphabet.git ~/ && cd ~/alphabet \ && source alphabet.profile
```

## Development

```sh
$ source alphabet.profile && source  ~/[.zshrc|.bash_profile]
```
